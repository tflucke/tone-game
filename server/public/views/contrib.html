How can I contribute?

If you like this project and want to help, there are several ways anyone can make it better.

Voice Recordings

Fluent Vietnamese speakers can help by expanding the catalog of recordings. Please send high quality voice recordings to this email, with how you would like to be credited. In order to eliminate the ability for players to determine the correct spelling from process of elimination, make sure to submit recordings in sets of all six tones and try to keep recording quality consistent.

Code

Programmers can add features from the wish list to this project on github. To contribute make a fork of the project, and submit a pull request with the new improvements.

Wish List

  • Permalink to a word
  • Expanded word catalog
  • More voice recordings
  • Show translations after correct answer
  • User progress tracking
  • Adjustable difficulty
