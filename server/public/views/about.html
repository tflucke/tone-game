What is this?

This is a game I created to help myself and others learn to pronounce Vietnamese properly.

I was practicing Vietnamese on Duolingo when I realized I would often not even think about the tone of the words. I was able to translate each word based on the context and process of elimination rather than actually hearing the tone. While this was good enough to make progress, this realization was dissatisfying for someone who wanted to learn how to pronounce words properly.

I created this game to remove all other variables and isolate the key skill I felt I was missing: the ability to recognize a word by tone alone.

The game will prompt the player with a recording of a Vietnamese word and ask the player to select the correct spelling. Six different spellings will be presented, each with the same basic spelling but each with a different tonal inflection. Thus, the only way to select the correct word is to correctly identify the tone.
